很显然，我们只要将需要删去的部分翻到一个字符串里，strip() 方法会从字符串两头开始依次搜索符合条件的字符并删去，直到出现不符合条件的字符就停止，然后返回修剪后的字符串副本

‼️注意：strip() 不会对原有的字符串进行操作，而是将该字符串复制一份后，对这个副本进行操作，所以如果想继续使用修改过后的字符串，就一定要使用变量去接收！

在一般情况下，如果我们不给 strip() 方法传递任何参数，直接使用 str.strip() ，strip() 方法会默认删除字符串开头和末尾的空格，像下面这样：

“查” Find

字符串其实没有实际意义上的“查询”一说，一般字符串的“查”会用于截取字符串或者按条件筛选字符串这两种情况，下面就来分开讲一讲：

  1. 字符串切片
    所谓切片，就是从字符串中把某一部分单独拿出来。这里就需要引入一个概念：索引（index），其实并不复杂，说白了就是字符串中每个字符在字符串中的位置序号。不过需要注意的是，Python当中的索引一律从0开始，也就是说第一位序号是0，第二位序号是1，以此类推。

    下面来看下如何利用索引对字符串进行切片：

    有没有发现，明明需要的索引是[0:4]，在写的时候却是[0:5]？这是因为切片的索引遵循算头不算尾这一规则，所以如果我们需要取到第五位（索引为4），那写法上就要写到索引的下一位（即索引为5）

    其实可以理解为：指定头尾索引，从 头索引起，截取至尾索引的前一位
